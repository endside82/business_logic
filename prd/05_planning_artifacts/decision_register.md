## 2. 결정 기록 양식

| 항목 | 내용 |
|---|---|
| 결정 ID | D-00 |
| 결정일 | YYYY-MM-DD |
| 결정자 | 이름/역할 |
| 결정 내용 | 한 문장으로 작성 |
| 근거 | 사용자 피해, 운영 비용, 기술 제약, 법무 검토 등 |
| 영향을 받는 문서 | 관련 PRD 파일 또는 기능 ID |
| 재검토 조건 | 지표/정책/출시 범위 변경 시 |
