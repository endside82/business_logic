# MVP 범위 매트릭스

<!-- supporting-doc-status: 2026-05-18 -->

> 문서 상태: **보조 문서**. 기능별 현재 계약, source trace, Gap/Risk 판단은 [PRD_MIGRATION_STATUS.md](../PRD_MIGRATION_STATUS.md)와 각 기능 PRD를 우선한다. 이 문서는 인벤토리, 정책, QA, 기획 운영 기준을 보조하며, 기능 세부 판단은 [FEATURE_PRD_STANDARD.md](../FEATURE_PRD_STANDARD.md) 기준으로 재확인한다.

## 1. 우선순위 판단 기준

| 우선순위 | 기준 |
|---|---|
| P0 | 첫 사용자가 핵심 가치를 경험하기 위해 반드시 필요 |
| P1 | 핵심 경험을 안정화하거나 반복 사용을 높임 |
| P2 | 운영 효율, 고급 기능, 성장 이후 확장 |
| 보류 | 법무, 운영, 기술, 정책 결정 없이는 출시 위험이 큼 |

## 2. 영역별 MVP 검토표

| 영역 | P0 후보 | P1 후보 | 보류 후보 |
|---|---|---|---|
| 인증 & 온보딩 | 이메일/소셜 로그인, 온보딩, 관심사 | 계정 연결 해제 | 고급 계정 복구 정책 |
| 홈 피드 | 홈 메인, 추천 카드 진입 | 더보기/필터 | 고도화 추천 |
| 이벤트 | 탐색, 상세, 생성, 신청, 참석 | 체크인, 사진, 위시리스트 | 복잡한 승인/대기열 정책 |
| 클럽 | 탐색, 상세, 가입, 멤버 관리 | 게시판, 공지, 사진첩 | 기금, 구독, 출금 |
| 검색 | 키워드 검색, 필터 | 자동완성, 저장 검색 | 개인화 검색 |
| 결제 & 지갑 | 지갑, 충전, 결제, 환불 | 결제수단, 거래내역 | 수익 대시보드, 자동충전 |
| 모임 정산 | 생성, 항목, 납부, 상태 요약 | 독촉, 이의제기 | 복잡한 환불/감사 |
| 플랜 마켓 | 작성, 발행, 탐색, 구매 | 컬렉션, 리뷰 | 크리에이터 통계 |
| 프라이빗 데이팅 | 인증, 프로필, 후보자, 매칭 | 채팅, 만남 제안 | 안전 고급 기능 |
| 캘린더 | 통합 보기, 상세 이동 | 가용시간 | 반복 가용성 |
| 리뷰 & 신고 | 리뷰 작성, 신고 | 신뢰점수 | 취향 데이터 고도화 |
| 알림 | 목록, 읽음, 권한 안내 | 카테고리, 방해금지 | 다중 기기 관리 고도화 |
| 프로필 & 설정 | 프로필, 주소, 선호태그 | 데이터 내보내기 | 삭제/비활성화 세부 정책 |
| 위치 & 길찾기 | 길찾기, 주소 | 위치 공유 | 위치 프라이버시 대시보드 |

## 3. 사용 방법

- 이 표는 최종 범위가 아니라 첫 논의용 기준이다.
- P0로 올리는 기능은 반드시 기능 PRD의 수용 기준까지 함께 검토한다.
- 보류 기능은 결정 등록부에 사유와 재검토 조건을 남긴다.
