회계·복식부기

결제·정산·인출 흐름 뒤에서 모든 거래를 복식부기로 기록한다

  • 복식부기 원칙 — 충전·결제·환불·기금 인출 등 모든 돈의 이동이 빠짐없이 장부에 기록된다
  • 충전 단위 추적 — 충전할 때마다 별도 단위로 기록되고, 결제 시 오래된 것부터 차감, 환불 시 해당 단위별로 되돌린다
  • 역분개 — 정산이 취소되면 원 기록을 지우지 않고 반대 기록으로 되돌린다 (F07-03)
  • 거래 유형: 충전, 결제, 환불, 인출, 기부 등

읽는 법

  • 이 페이지는 인덱스다. 각 항목의 상세 동작과 분기는 해당 기능 PRD에서 본다.
  • “계획 단계”로 표시된 값은 정책으로는 정해졌지만 아직 제품에 반영되지 않은 항목이다. 인용 전 반드시 확인.
  • 법무 문구·UI 카피로 인용할 때는 항상 해당 기능 PRD에서 최종 값을 재확인한다.
