기능핵심 원칙
위치 공유직접 동의가 기본, 즉시 중지 가능, 공유 상태 항상 시각화

설계 시 4가지 질문

민감 기능을 추가·수정할 때 매번 묻는다.

  1. 동의가 명확한가 — “설정에 들어가서 켰다”가 아니라 사용자가 결과를 알고 동의했는가.
  2. 끄는 방법이 명확한가 — 끄는 경로가 1탭 거리에 있는가.
  3. 지금 켜져 있다는 사실이 보이는가 — 현재 상태가 본인 화면에서 보이는가.
  4. 실수로 켜졌을 때 되돌릴 수 있는가 — 신고·차단·삭제의 잘못 누름을 복구할 동선이 있는가.

수용 기준

  • 민감 기능을 켤 때만 화면을 만들지 말고, 끌 때 어디로 가야 하는지도 같은 자리에 둔다. 켜기와 끄기가 한 화면에서 모두 보여야 한다.
  • 한 화면에 “남에게 보이는 정보”와 “나만 보는 정보”가 섞이지 않게 영역을 나눠 표시한다.
  • 삭제·차단·신고·위치 공유 중지는 버튼을 누른 그 순간에 화면이 즉시 바뀌어야 한다. “처리 중”은 허용하지 않는다.
