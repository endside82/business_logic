레드 플래그

  • 삭제 예약과 즉시 비활성화 문구 혼동 — 사용자가 “지웠다”와 “감췄다”를 헷갈리지 않게 분명히 안내.
  • 데이터 내보내기를 즉시 다운로드라고 오해 — 비동기 처리임을 명시.
  • 주소 한도 초과 시 명확한 안내 없음 — “더 이상 추가할 수 없다”라고 정확히 알림.
  • 즉시 비활성화 시 미정산 처리 누락 — 호스트가 정산 중인데 비활성화하면 참가자 환불·납부 차단.
