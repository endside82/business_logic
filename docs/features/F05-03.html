F05-03. 검색 필터 적용

검색바 옆 필터 아이콘으로 카테고리·가격·거리·무료를 좁힌다. 필터 시트 안에서 “N개 결과 보기” 미리보기 카운트가 실시간으로 갱신되어 적용 전에 결과 규모를 확인할 수 있다.

필터 종류
4
시나리오
5+
GPS 캐시
5분

이 기능 안의 하위 항목

동작 1

카테고리 칩

동작 2

가격 듀얼 슬라이더

동작 3

거리 슬라이더(GPS 필요)

동작 4

무료 토글

동작 5

미리보기 카운트

사용자 시나리오

S1

카테고리 + 가격으로 좁히기

  1. 필터 시트에서 변경 시마다 카운트 미리보기
  2. 적용 → 본 검색 재호출
S2

거리 필터 + GPS 권한

  1. 5km 조정 시 권한 요청
  2. 허용 시 5km 이내 결과
  3. 거부 시 “5km 이내” 칩 표시되지만 결과는 전체(사일런트)
S3

무료 토글

  1. “무료 이벤트만” 스위치 ON
S4

초기화

  1. 활성 필터 칩 옆 초기화 → 모든 필터 해제
S5

GPS 캐시 재사용

  1. 5분 이내는 권한 다시 안 물음
