122개 기능 카탈로그

전 기능을 한 화면에서 검색·필터링한다. 키워드는 ID·이름·주 사용자·검산 포인트 전부에서 찾는다. 영향 영역(알림·돈·캘린더·위치·신뢰)으로도 좁힐 수 있어 “돈이 움직이는 기능만”과 같은 질문에 즉시 답할 수 있다.

ID 도메인 기능명·검산 포인트 주 사용자 영향 상태 시나리오

읽는 법

  • ID — F도메인번호-순번 형식. 다른 문서·이슈에서 이 ID로 참조한다.
  • 검산 포인트 — 시나리오 작성 시 절대 빠뜨리면 안 되는 조건 키워드.
  • 영향 — 이 기능이 변경되면 같이 바뀌는 횡단 영역. 회귀 검토의 기준.
  • 시나리오 수 — 정상·예외·복구 케이스 합계. 많을수록 QA 비용이 큰 기능.
