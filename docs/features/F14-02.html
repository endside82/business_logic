F14-02. 위치 공유 중지 (opt-out)

사용자가 위치 공유를 명시적으로 끈다. 폴링 종료 + 상태 UPDATE + 일행 화면에서 본인 마커 제거.

이 기능 안의 하위 항목

동작 1

opt-out 확인 다이얼로그

동작 2

폴링 종료

사용자 시나리오

S1

도착 후 끄기

S2

다른 화면 진입해도 유지

서버 계약

MethodPath
POST/api/v1/events/{eventId}/location/opt-out
