F04-01. 클럽 발견 · 탐색

전체 클럽 그리드를 카테고리·키워드로 탐색한다. 비로그인 사용자도 같은 화면을 볼 수 있고, 카드의 멤버십 배지(소유자/관리자/멤버)는 로그인 사용자에게만 노출된다. 가입 시도는 상세 화면(F04-02)에서 분기.

하위 동작
4
시나리오
6
게스트
허용

이 기능 안의 하위 항목

동작 1

그리드 조회

20개씩 페이지 단위. 게스트도 진입 가능.

동작 2

카테고리 칩

스포츠·스터디 등 칩 전환 시 0페이지 재조회.

동작 3

키워드 검색

IME 확정 시 재조회.

동작 4

멤버십 배지

본인 카드에 “소유자/관리자/멤버” 라벨.

사용자 시나리오 6개

S1

비로그인 카테고리 탐색

  1. 인증 없이 0페이지 조회
  2. 스포츠 칩 → 재조회
  3. 카드 탭 → 상세(F04-02)
S2

로그인 사용자 본인 클럽 찾기

  1. “러닝” 키워드 → 자기 클럽 카드에 “멤버” 배지
  2. 카드 탭 → “클럽 홈” CTA
S3

무한 스크롤 + 풀투리프레시

  1. 20건씩 누적, last=true 시 끝 인디케이터
  2. 위로 당김 → 0페이지 재조회
S4

잘못된 카테고리 문자열

  1. 400 CLUB_INVALID_CATEGORY
  2. 전체 카테고리 선택으로 복귀
S5

네트워크 오프라인

  1. 전체 화면 에러 + “다시 시도”
  2. 네트워크 복구 후 정상
S6

FAB 권한 분기

  1. 로그인 시 FAB → /clubs/create
  2. 비로그인은 FAB 숨김
