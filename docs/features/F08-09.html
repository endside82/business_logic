F08-09. 마켓 검색

사용자가 찾을 단어를 가지고 들어왔을 때를 위한 면이다. 마켓 메인이 발견을 위한 화면이라면, 본 화면은 "한강", "와인" 같은 의도가 분명한 키워드에 빠르게 결과를 내놓는다. 외부 링크로 키워드가 박힌 채 들어와도 같은 결과를 보게 되고, 끝까지 스크롤하면 다음 묶음이 자연스럽게 이어 붙는다.

마켓 메인은 "무엇이든 한 번 보자"의 면이고, 검색은 "찾을 게 있다"의 면이다. 두 흐름이 한 화면에 섞이면 사용자는 둘 중 하나의 자유를 잃는다. 그래서 검색은 별도 화면으로 분리되어 — 마켓 메인 상단 검색 바를 누르면 이 화면으로 빠진다.

이 화면이 다루는 두 가지 진입

진입 1

마켓 메인에서 키워드 입력

마켓 메인 상단 검색 바에 단어를 입력하고 엔터를 치면 이 화면으로 빠지면서 즉시 결과가 표시된다. 검색 입력칸에 같은 단어가 그대로 들어 있어 키워드를 확인하면서 결과를 본다.

진입 2

외부 링크 직접 진입

홈 추천이나 외부 공유 링크에 키워드가 박혀 있다. 그 링크로 본 화면을 열면 검색칸에 키워드가 자동으로 채워지고 결과가 한 번에 로드된다. 사용자가 별도 타이핑 없이 키워드를 확인할 수 있다.

사용자 시나리오

기본 흐름

"한강" 한 단어로 결과를 찾는다

한강 데이트 코스를 찾는 사용자. 마켓 메인 검색 바에 "한강"을 입력하고 엔터. 이 화면으로 빠지면서 상단에 "한강 검색 결과 12건" 카운트가 뜨고 두 줄짜리 그리드로 카드가 나열된다. 네 번째 카드를 누르면 마켓 아이템 상세로 들어간다.

무한 스크롤

결과 끝까지 둘러본다

결과 20장을 본 사용자가 끝까지 스크롤하면 다음 묶음 12장이 자연스럽게 이어 붙는다. 더 이상 결과가 없으면 추가 호출이 일어나지 않는다.

빈 결과

잘못된 키워드 입력

실수로 "asdfqwerty" 같은 단어를 친다. 결과 0건. 화면 가운데에 ""asdfqwerty"에 대한 결과가 없습니다 / 다른 키워드로 검색해 보세요" 안내가 뜬다. 사용자가 검색 바에 다른 단어를 다시 친다.

첫 진입 안내

검색 화면을 빈 채로 직접 들어왔다

아무 키워드 없이 검색 화면 URL로 직접 들어온 사용자. 검색칸이 자동으로 활성화되고 가운데에 "검색어를 입력해 주세요 / 플랜 제목, 설명, 태그에서 검색합니다" 안내가 뜬다. 사용자가 키워드를 입력하면 결과가 로드된다.

네트워크 실패

검색 도중 끊긴다

키워드를 입력했지만 네트워크가 끊겨 결과가 안 온다. 화면에 에러 상태와 "재시도" 버튼이 뜬다. 사용자가 누르면 같은 키워드로 다시 시도한다.

공백 가드

공백만 친 채로 엔터

공백만 입력하고 엔터를 치면 검색 호출이 일어나지 않는다. 화면은 변하지 않고 안내 상태 그대로다.

외부 진입

외부 링크에서 키워드와 함께 진입

다른 채널에서 공유받은 링크가 "?keyword=Boardgame"을 가리키고 있다. 화면이 열리는 동시에 검색칸에 "Boardgame"이 채워지고 결과가 자동으로 로드된다. 사용자는 별도 타이핑 없이 결과를 확인하고, 필요하면 키워드를 수정해서 다시 검색한다.

이 흐름이 남기는 것 — 외부 마케팅 채널과 마켓 검색이 매끄럽게 이어지는 길. 같은 키워드의 결과를 어디서 들어오든 같게 보장.

운영자가 알아야 할 것

검색 결과의 신뢰가 마켓 발견의 신뢰다
  • 현재 검색은 키워드 한 가지만 사용한다. 서버는 카테고리·통화·종류·가격대·정렬을 받을 수 있지만, 화면에서는 키워드만 보내고 있다. 결과 품질이 떨어진다면 이 옵션들을 노출하는 것이 첫 개선 카드.
  • 결과 카운트는 화면에 누적된 카드 수다. 서버가 알려 주는 전체 건수가 아니다 — 무한 스크롤로 카드를 받을 때마다 숫자가 늘어난다. 사용자가 "12건"으로 봤다가 스크롤하면 "32건"으로 바뀌는 인지 차이가 생긴다.
  • 외부 링크 진입 시 키워드는 대소문자·공백까지 그대로 검색 입력에 채워진다. 사용자가 입력 자체를 다시 손볼 수 있어, 잘못된 진입에서도 회복 경로가 있다.
  • 검색 트렌딩 집계는 본 단위에서 다루지 않는다. 인기 검색어·자동 완성 같은 후속 기능은 통합 검색 도메인으로 이관 검토가 필요하다.
