F05-05. 저장된 검색

자주 쓰는 검색(키워드 + 필터)을 한 번에 다시 실행. 사용자당 최대 10개, 알림 토글 옵션.

한도
10개
시나리오
5

이 기능 안의 하위 항목

동작 1

저장

이름·알림 토글, 검색 파라미터 JSON 직렬화.

동작 2

목록 조회

동작 3

실행

useCount++ + 검색 메인 복원.

동작 4

수정·삭제

사용자 시나리오 5개

S1

저장 → 실행

  1. 북마크 → 이름·알림 입력
  2. 다음에 카드 탭 → 키워드+필터 복원
S2

10개 초과

  1. INVALID_INPUT, “저장에 실패했습니다” 토스트
S3

알림 토글 갱신

S4

삭제

S5

idle 상태 진입점

  1. 저장된 검색을 검색 메인 idle 화면에 표시
