F08-14. 플랜 마켓 환불

마켓 아이템·번들·플랜 직접 구매분에 대한 환불 흐름. 구매자가 신청하고 판매자가 승인/거절하며, 판매자 무응답에는 자동 처리가 붙는다. 1인 크리에이터 판매자가 많아 무응답·지연 응답이 흔하므로 사용 흔적이 없는 7일 무응답 건은 자동으로 환불 승인되고, 사용 흔적이 있는 건만 운영자 검토로 올라간다. 거절에 동의 못 하는 구매자는 분쟁을 열어 운영자 중재로 갈 수 있다.

설계 의도 — 판매자 무응답이 구매자를 인질로 잡지 않는다

1인 크리에이터 판매자가 많아 무응답·지연 응답이 잦다. 판매자 결정만 기다리지 않고 7일 자동 처리를 깐다 — 사용 흔적이 없으면 자동 승인(환불), 사용 흔적이 있으면 운영자 검토로 올라간다. 판매자가 추가 자료를 요청해 타이머를 한 번 멈출 수 있는 안전장치도 있다.

사용자 시나리오

구매자 신청

"산 플랜이 설명과 다르다"

구매자가 보유함에서 카드 누르고 "환불 신청". 사유 분류(품질·설명 불일치·실수·하자·기타) + 메모 입력. 신청 즉시 카드는 잠긴다(사용·다운로드·이벤트 연결 불가).

이 흐름이 남기는 것 — 신청 직후 잠금은 환불 진행 중 추가 사용으로 인한 분쟁을 방지한다.

판매자 승인

"판매자가 사유 보고 인정"

판매자가 대시보드에서 신청 확인 후 "승인". 즉시 환불이 발사되고 구매자 지갑으로 결제 금액 복원. 카드는 보유함에서 제거되거나 "환불 완료" 표기.

이 흐름이 남기는 것 — 판매자가 빠르게 결정 내릴 수 있는 환경, 분쟁으로 이어지지 않게.

판매자 거절

"판매자가 거절 사유 입력 후 거절"

판매자가 "거절" 누르고 사유 입력. 카드 잠금 해제, 구매자는 다시 사용 가능 상태로 복귀. 거절 알림. 구매자는 24시간 후 1회 추가 신청 가능.

이 흐름이 남기는 것 — 잘못된 신청이 즉시 환불로 이어지지 않게, 판매자 권한 보호.

자동 환불

"판매자 7일 잠수 + 사용 흔적 없음"

판매자가 7일 동안 응답하지 않고 구매자가 다운로드·이벤트 연결·활성화를 한 적 없으면 자동으로 환불 승인. 즉시 환불 진행.

이 흐름이 남기는 것 — 무책임한 판매자가 구매자를 무기한 잡고 있지 않게 한다.

운영자 검토 에스컬레이션

"7일 잠수인데 사용 흔적 있음"

구매자가 한 번이라도 다운로드·이벤트 연결을 했으면 자동 환불 불가. 운영자 검토 큐로 이관되어 운영자가 사용 흔적·금액·이력을 보고 결정.

이 흐름이 남기는 것 — 사용했다 환불 받는 식의 무료 사용 패턴 차단.

분쟁

"거절에 동의 못 한다"

구매자가 거절로부터 7일 이내 1회 분쟁 신청. 운영자 큐로 진입 → 운영자가 거절 유지 또는 강제 환불 선택.

이 흐름이 남기는 것 — 판매자 거절을 끝점으로 두지 않고 사용자가 항소할 수 있는 경로.

추가 자료 요청

"판매자가 사진 한 장 더 달라고 요청"

판매자가 "추가 자료 요청" 누르면 구매자에게 자료 요청 알림. 7일 자동 타이머는 멈춤. 구매자가 응답하면 다시 판매자 결정 대기로, 응답 안 하면 신청 만료.

이 흐름이 남기는 것 — 정보 부족으로 거절되는 일을 줄이고, 판매자가 자료를 보고 판단할 수 있게.

운영자가 알아야 할 것

사용 흔적·자동 처리·재신청
  • "사용 흔적"의 정의 — 다운로드 횟수, 이벤트 연결, 활성화 시각. 이 중 하나라도 있으면 자동 환불 불가, 운영자 검토.
  • 거절 후 24시간 1회 재신청 — 같은 구매에 무한 신청을 막기 위함.
  • 분쟁은 거절로부터 7일 이내 1회 — 단발성. 분쟁이 끝나면 결과 확정.
  • 운영자 강제 환불은 별도 라벨 — 정산·노출 패널티 누적에 영향(별도 정책).
  • 구매 잠금 동안 사용 불가 — 환불 신청 즉시 카드는 잠기고, 결정이 날 때까지 사용·다운로드 막힘.

레드 플래그

자주 발생하는 케이스
  • 환불 후 유료/무료 분리 미보존 — 현재는 환불 시 전액이 유료 포인트로 복원된다. 원결제의 유료/무료 비율 보존은 후속.
  • 매출 귀속(F08-15)과 의존 — 환불은 크리에이터 매출 원장을 잠그거나 역전시키므로 F08-15가 정확해야 환불도 정확하다.
  • 추가 자료 요청 후 기한 만료 — 사용자가 응답 안 하면 환불 신청이 만료된다(거절 아님). 사용자에게 명확히 안내 필요.
  • 판매자가 잠수타면 7일 카드 잠금 — 사용자 입장에서 답답할 수 있어 신청 시 안내 필수.
