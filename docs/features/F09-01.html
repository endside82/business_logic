F09-01. 본인 인증 (Toss)

데이팅 진입에 필요한 통신사 본인인증. Toss WebView로 진행되며, 인증 결과(이름·생년월일·성별·CI)는 AES 재암호화되어 저장. 1년 유효.

이 기능 안의 하위 항목

동작 1

상태 조회

동작 2

인증 요청 + Toss WebView

동작 3

결과 검증·저장

동작 4

만료 갱신
