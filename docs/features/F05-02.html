F05-02. 자동완성 서제스트

검색바에서 입력 중 카테고리·이벤트 제목 후보를 추천한다. 디바운스 + 최소 2글자 정책으로 불필요한 네트워크 트래픽을 줄이고, 카테고리 매칭이 가장 위에 노출된다.

  2. “등산” → 디바운스 후 호출 → 카테고리 + 이벤트 제목 후보
