F07-09. 선입금 결제 · 환불 · 환불 규정

호스트가 행사 전 단계별 환불률(예: 7일/3일/1일/당일)을 설정하고, 참가자가 선입금한다. 취소 시점에 따라 환불률 자동 계산.

이 기능 안의 하위 항목

동작 1

환불 규정 설정

동작 2

선입금 결제(POINT/카드)

동작 3

호스트 확인(계좌이체)

동작 4

취소 시점별 환불 계산

사용자 시나리오

S1

호스트 규정 4단계 저장

S2

참가자 POINT 선입금

S3

호스트 계좌이체 확인

S4

참가자 D-2 취소 → 30% 환불

S5

당일 취소 → 0% 환불

S6

호스트 취소 → 100% 환불

MethodPath

POST/api/v1/events/{eventId}/prepayments/{id}/cancel
