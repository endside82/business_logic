F07-01. 모임 정산 생성

종료된 이벤트의 비용을 호스트가 정산 단위로 생성한다. 모드(호스트 일괄 수금/멤버 각자 부담), 총액, 참가자 선택 후 DRAFT로 만들어진다. 활성화는 F07-03에서.

이 기능 안의 하위 항목

동작 1

정산 생성 폼

모드·총액·참가자 선택.

동작 2

첫 항목 자동 등록

EQUAL 분할로 share 자동 생성.

동작 3

이전 정산 복제

같은 멤버 정기 모임 빠르게.
