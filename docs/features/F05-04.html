F05-04. 최근 검색어

사용자 검색 키워드를 Redis에 자동 기록한다. 최근 20개를 유지, 90일 TTL. 같은 키워드는 최상단으로 이동.

최대 보관
20개
TTL
90일
시나리오
5

이 기능 안의 하위 항목

동작 1

자동 기록

검색 호출 시점 서버가 LPUSH + LTRIM + EXPIRE.

동작 2

최근 검색어 표시

idle 상태에 첫 10건.

동작 3

개별 삭제 / 전체 삭제

사용자 시나리오 5개

S1

자동 기록 + 재검색

  1. 탭한 키워드 → 최상단 이동
S2

빈 기록

  1. 섹션 자체 미노출
S3

중복 입력

  1. 기존 항목 LREM 후 LPUSH → 최상단
S4

21개째 입력

  1. LTRIM 0 19로 가장 오래된 항목 제거
S5

전체 삭제

  1. 섹션 초기화
