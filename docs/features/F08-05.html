F08-05. 플랜 발행

DRAFT → PUBLISHED 전환. 체크리스트(제목·블록·설명·커버·가격)가 모두 통과해야 발행 가능. 발행 후 판매 중지/재발행 분기.

이 기능 안의 하위 항목

동작 1

발행 체크리스트

동작 2

가격·설명·커버 마지막 편집

동작 3

발행 / 판매 중지

사용자 시나리오

S1

첫 발행

S2

체크리스트 미통과

S3

판매 중지 → UNPUBLISHED

S4

재발행
