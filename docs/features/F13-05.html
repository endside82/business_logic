F13-05. 데이터 내보내기 (Export)

사용자가 본인 데이터 사본을 요청. 백그라운드 처리 후 다운로드 가능 → 만료 후 자동 삭제.

이 기능 안의 하위 항목

동작 1

요청 + 상태(PROCESSING/READY/FAILED)

동작 2

다운로드

동작 3

재요청 / 실패 재시도

사용자 시나리오

S1

처음 요청

S2

처리 중 상태 표시

S3

S4

실패 → 다시 요청
