F08-04. 블록 드래그 재정렬 · 계층 이동

트리 평탄화 후 드래그앤드롭으로 순서 변경. 부모 변경된 블록만 별도 move 호출, 같은 부모 안의 순서 변경은 reorder 한 번으로.

이 기능 안의 하위 항목

동작 1

로컬 평탄화

동작 2

드래그 인터랙션

동작 3

부모 변경 → move

동작 4

순서 변경 → reorder

사용자 시나리오

S1

같은 부모 순서 변경

S2

다른 부모로 이동

S3

저장 실패 롤백
