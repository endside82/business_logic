F08-06. 마켓 아이템 관리

발행된 플랜을 마켓 노출용 아이템(가격·이미지·재고)으로 등록. READY → ON_SALE → CLOSED 상태 전이.

이 기능 안의 하위 항목

동작 1

아이템 등록(READY)

동작 2

발행(ON_SALE)

동작 3

가격·재고 수정

동작 4

판매 종료(CLOSED)

사용자 시나리오

S1

아이템 등록

S2

S3

S4

재고 소진 자동 종료
