F11-01. 이벤트 리뷰 작성

종료된 이벤트(ATTENDING + 7일 이내)에 한해 별점 + 본문을 1회만 작성. 호스트 본인은 작성 불가.

이 기능 안의 하위 항목

동작 1

리뷰 작성 폼

동작 2

별점 + 본문

동작 3

1회 제한 검증

사용자 시나리오

S1

참석자 리뷰 작성

S2

중복 작성 차단

S3

호스트 본인 작성 시도

S4

7일 초과 작성 시도

서버 계약

MethodPath
POST/api/v1/events/{eventId}/reviews
