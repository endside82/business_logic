F13-03. 다중 주소 관리

집·회사 등 라벨로 구분된 주소 N개(최대 10) 관리. 기본 주소는 추천·길찾기 기본값. Daum 우편번호 + 역지오코딩 결합.

이 기능 안의 하위 항목

동작 1

주소 추가(라벨·검색·좌표)

동작 2

기본 주소 전환

동작 3

수정·삭제

사용자 시나리오

S1

집 주소 추가 + 기본 지정

S2

10개 한도

S3
