F04-12. 클럽 이벤트 · 캘린더

클럽 전용 이벤트를 생성·게시·참석한다. 게시 시점에 가입 순서대로 정원만큼 자동 참석 처리되고, 캘린더 탭에서 월별 이벤트 도트로 한눈에 본다. 일반 이벤트(F03)와 같은 상태 머신을 따르지만 멤버 자동 참가가 클럽만의 특징.

하위 동작
5
시나리오
7+

이 기능 안의 하위 항목

동작 1

클럽 이벤트 생성

DRAFT INSERT.

동작 2

게시(자동 참가)

가입 순서대로 정원만큼 ATTENDING 일괄 INSERT + 알림.

동작 3

캘린더 조회

월별 도트 + 날짜별 카드.

동작 4

참석/대기/취소

대기열 자동 승급 포함.

동작 5

이벤트 취소(유료 환불)

실패 시 고객센터 안내.

사용자 시나리오 7개

S1

게시 + 자동 참가

  1. DRAFT → OPEN
  2. 가입 순서대로 20명 ATTENDING
  3. 본인 제외 알림
S2

캘린더 → 참석

  1. 월별 도트 → 날짜 셀 탭 → 카드 → 상세 → ATTENDING INSERT
S3

정원 가득 → 대기

  1. WAITING 순번 부여
S4

취소 → 대기 승급

  1. 다음 대기자 자동 ATTENDING + 알림
S5

유료 이벤트 취소

  1. 참가자 자동 환불, 실패는 고객센터 안내
S6

DRAFT만 수정·삭제

  1. OPEN 수정·삭제는 EVENT_INVALID_STATUS_TRANSITION
S7

BUSINESS 구독 만료 차단

  1. 게시·참가 모두 CLUB_SUBSCRIPTION_EXPIRED
