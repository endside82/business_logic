F08-13. 구매 플랜 이벤트 생성 · 리뷰 작성

구매한 플랜으로 이벤트를 만들거나(F03 연동) 리뷰를 쓴다. 플랜 메타가 이벤트 생성 폼에 자동 prefill.

이 기능 안의 하위 항목

동작 1

플랜 → 이벤트 생성(prefill)

동작 2

리뷰 작성 + 별점

동작 3

본인 리뷰 수정·삭제

사용자 시나리오

S1

이벤트 생성 prefill

S2

S3

S4

크리에이터 평점 영향
