F14-05. 이벤트 길찾기 (경로 + 참석자 거리)

현재 위치 또는 등록 주소에서 이벤트 장소까지 경로·이동 시간·교통수단 안내. 외부 지도 앱 연동.

이 기능 안의 하위 항목

동작 1

현재 위치 기반

동작 2

등록 주소 기반

동작 3

참석자 거리 표시

동작 4

외부 지도 앱 열기

사용자 시나리오

S1

현재 위치 → 모임 장소

S2

등록 주소 선택

S3

외부 지도 앱(카카오/네이버 지도)

S4

권한 거부
