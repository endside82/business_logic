F17-08. 환불 (pro-rata · 트리거별 · FAILED_REFUND)

FIXED 모임 환불은 진행 세션 차감 pro-rata 산식 + 3 트리거 분기. 모두 amount × (N − consumed) / N 골격을 쓰고 consumed 정의만 다르다. 100원 단위 floor, 0 clamp.

산식

total = meeting.totalSessionCount
consumed = (트리거별)
remaining = max(0, total - consumed)
raw = paymentAmount × remaining / total (RoundingMode.DOWN)
refundAmount = floorToHundred(raw)
