F04-08. 클럽 게시판 · 게시글 CRUD

자유게시판·공지·이벤트 게시판 등 보드별 게시글을 작성·수정·삭제·고정한다. 작성은 보드 권한에 따라, 수정은 작성자 본인만, 삭제는 작성자 + ADMIN까지 허용된다.

하위 동작
5
시나리오
8
이미지 한도
10장/글

이 기능 안의 하위 항목

동작 1

작성

이미지 사전 서명 PUT → 게시글 INSERT.

동작 2

수정(작성자만)

이미지 sync로 추가/삭제/순서 갱신.

동작 3

삭제(작성자·ADMIN)

soft delete.

동작 4

고정 토글(ADMIN)

상단 고정 섹션 노출.

동작 5

조회수 증가

상세 진입 시 원자적 +1.

사용자 시나리오 8개

S1

자유게시판 작성

  1. 이미지 2장 첨부 → POST → 상세 push
S2

NOTICE 작성 시도(일반 멤버)

  1. FAB UI 가드 + 우회 시 BOARD_NO_WRITE_PERMISSION
S3

고정 토글

  1. ADMIN이 고정/해제, 상단 섹션 노출
S4

본인 수정

  1. PUT + 이미지 sync
S5

ADMIN의 타인 글 수정 시도

  1. POST_NO_EDIT_PERMISSION(서버에서도 차단)
S6

ADMIN의 타인 글 삭제

  1. soft delete
S7

조회수 증가

  1. 매 진입마다 +1
S8

삭제된 글 진입

  1. POST_NOT_FOUND + 안내
