F09-03. 후보자 스와이프 · 매칭 액션

후보자 카드를 LIKE/PASS로 평가. 양쪽 LIKE 시 즉시 매치 + 채팅방 생성. 차단된 사용자는 양방향 풀에서 제외.

동작 2

LIKE/PASS 액션

동작 3

매치 생성 + 채팅방 INSERT
