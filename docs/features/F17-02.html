F17-02. 정기모임 상세 조회

모임 본체 + 세션 요약 + 조회자 컨텍스트를 단일 엔드포인트에서 받는다. DRAFT는 호스트만, 나머지는 optional auth로 누구나 조회 가능. status × viewer 매트릭스로 하단 CTA가 정해진다.

isHost JSON key는 host

VO 필드는 isHost지만 Jackson이 is 프리픽스를 제거해 JSON 키는 host로 직렬화된다. Flutter는 @JsonKey(name: 'host')로 명시 매핑한다.

status × viewer CTA 매트릭스

statushostmemberguest

서버 계약

  • GET /api/v1/regular-meetings/{id} — optional auth. DRAFT는 hostUserId 본인만, 그 외 REGULAR_MEETING_NOT_FOUND(2700001)로 마스킹.
  • GET /api/v1/regular-meetings/{id}/events?includeHistory={bool} — CANCELED 포함 옵션.
  • GET /api/v1/regular-meetings/{id}/members/me — 인증 필수, 없으면 404 (silent).
