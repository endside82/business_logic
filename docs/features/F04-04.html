F04-04. 멤버 관리

클럽 멤버의 역할 변경과 추방을 다룬다. OWNER는 ADMIN을 임명·해임할 수 있고, ADMIN은 일반 MEMBER만 추방할 수 있다. OWNER 본인은 어떤 경로로도 추방·변경되지 않는다.

역할
3
OWNER·ADMIN·MEMBER
시나리오
9

권한 매트릭스

실행자\\대상OWNERADMINMEMBER
OWNER—(이전만 가능)역할 변경·추방 가능역할 변경·추방 가능
ADMIN차단차단추방 가능(역할 변경 불가)
MEMBER차단차단차단

이 기능 안의 하위 항목

동작 1

멤버 목록 + 역할 필터

탭: 전체/관리자/멤버. 페이징.

동작 2

역할 변경(OWNER 전용)

ADMIN ↔ MEMBER 토글. OWNER는 옵션 자체 없음.

동작 3

추방

BUSINESS는 가입비 항상 환불(KICKED 환불 규칙).

사용자 시나리오 9개

S1

OWNER가 ADMIN 승격

  1. 역할 변경 시트 → ADMIN → 저장
  2. 대상에게 권한 부여
S2

ADMIN의 역할 변경 시도

  1. UI 항목 숨김, 우회 시 403
S3

OWNER 역할 변경 차단

  1. UI에 OWNER 옵션 없음
  2. 우회 시 CLUB_CANNOT_CHANGE_OWNER_ROLE
S4

ADMIN이 MEMBER 추방

  1. 가입비 환불 + DELETE + memberCount--
  2. 대상에게 CLUB_MEMBER_KICKED 알림
S5

ADMIN→ADMIN 추방 시도

  1. UI 항목 없음, 우회 시 403 CLUB_NO_PERMISSION
S6

OWNER→ADMIN 추방

  1. OWNER는 모든 ADMIN 추방 가능
S7

OWNER 자기 추방 시도

  1. 본인 카드에 ⋮ 미표시
  2. 우회 시 403
S8

역할 필터

  1. “관리자” 탭 → role=ADMIN 쿼리
S9

비멤버 진입

  1. 라우터 차단
  2. 우회 시 401/403

서버 계약
