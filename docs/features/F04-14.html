F04-14. 클럽 기금에 기부하기

FREE 클럽 멤버가 본인 지갑에서 클럽 기금으로 송금한다. 기부 + 지갑 차감 + 회계 분개가 한 트랜잭션. 7일 이내 본인 취소만 허용되며, 인출 PENDING으로 가용 잔액이 부족하면 취소가 막힌다.

하위 동작
3
기부·내역·취소
시나리오
8
취소 기한
7일

이 기능 안의 하위 항목

동작 1

기부 결제

지갑 차감 + 기금 적립 + 분개 한 트랜잭션.

동작 2

내역 조회

최신순, 본인/타인 기부 표시.

동작 3

본인 취소(7일)

역분개 + 지갑 환불. 가용 기금 부족 시 차단.

사용자 시나리오 8개

S1

5만원 기부

  1. 확인 다이얼로그 → POST
  2. 지갑 차감 + 기금 적립 + 분개 한 번에
S2

잔액 부족

  1. UI 사전 차단
  2. 우회 시 INSUFFICIENT_BALANCE + 충전 안내
S3

BUSINESS 기부 시도

  1. UI에 CTA 없음, 우회 시 CLUB_DONATION_NOT_ALLOWED
S4

비활성 클럽

  1. CLUB_INACTIVE
S5

본인 7일 내 취소

  1. 비관적 락 + 가용 잔액 검증
  2. 지갑 환불 + 역분개
S6

7일 초과 취소 시도

  1. 버튼 비활성, 우회 시 CLUB_DONATION_CANCEL_PERIOD_EXPIRED
S7

타인 기부 취소 시도

  1. CLUB_NO_PERMISSION
S8

기금 가용 부족

  1. PENDING 인출로 가용 부족 시 취소 차단

서버 계약

MethodPath
POST/api/v1/clubs/{id}/donations
GET/api/v1/clubs/{id}/donations?page=&size=
POST/api/v1/clubs/{id}/donations/{donationId}/cancel
