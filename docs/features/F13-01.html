F13-01. 내 프로필 (마이페이지)

하단 탭 마이페이지를 눌렀을 때 보이는 첫 화면. 사용자의 정체성(닉네임·자기소개·사진)과 활동 요약(지갑·내 이벤트·내 클럽·신뢰점수), 그리고 프로필·주소·태그·정산·기기·프라이버시까지 하위 메뉴 진입점을 한 화면에 모은다. 본 단위 자체는 직접 데이터를 바꾸지 않고, "지금 내가 어떤 상태인지"를 한눈에 보여 주고 다음 화면으로 보내는 출구 역할만 한다. 계정 삭제가 진행 중이면 상단에 남은 일수 배너가 따라붙는다.

설계 의도 — "내 상태를 보는 곳"과 "내 데이터를 바꾸는 곳"을 분리

마이페이지는 사용자가 자기 정보를 직접 수정하는 자리가 아니다. 자기 신원과 활동을 빠르게 확인하고, 바꿔야 할 게 있으면 그 화면으로 한 번 더 들어가게 한다. 한 화면에 편집까지 몰아 두면 "잘못 눌러서 바뀐다"는 사고가 늘기 때문에, 본 화면은 의도적으로 조회와 라우팅만 책임진다. 지갑 잔액·이벤트 수·클럽 수·신뢰점수는 각각 다른 도메인이 채워 주는 값이라, 한 자리가 비어도 다른 자리는 정상으로 표시된다.

이 화면이 보여 주는 네 묶음

정체성

프로필 카드

원형 사진(없으면 기본 아이콘), 닉네임, 자기소개 2줄 말줄임. 자기소개가 비어 있으면 자리만 두고 채우지 않는다 — "자기소개를 입력하세요" 같은 안내 문구로 메우면 사용자가 자기 상태를 잘못 인지한다.

활동 요약

2×2 요약 카드

지갑·다가오는 이벤트 수·참여 클럽 수·신뢰점수. 각각 다른 도메인이 채운다. 값이 아직 안 들어왔으면 라벨만 두고 비워 두며, 들어오는 즉시 자연스럽게 채워진다. "카드마다 로딩 스피너"가 같이 도는 화면은 정책상 만들지 않는다.

하위 메뉴

설정 진입점 13개

프로필 편집·주소·태그·정산 계좌·정산 히스토리·기기·데이터 프라이버시·캘린더·내 리뷰·찜한 이벤트·앱 설정·고객센터·로그아웃. 본 화면은 이 목록을 한 자리에 모아 두고, 사용자가 누르면 해당 화면으로 보내는 일만 한다.

계정 상태 배너

삭제 진행 배너

30일 유예 삭제가 진행 중일 때만 화면 최상단에 나타난다. "계정 삭제 요청 진행 중 — 남은 일수" 표시와 "관리" 버튼으로 데이터 프라이버시 화면으로 빠진다. 취소되거나 진행 상태가 아니면 자리 자체가 없다.

사용자 시나리오

기본 진입

지금 내 상태를 확인하고 다음 작업으로 이동

마이페이지를 연다. 프로필 카드와 요약 4개가 거의 동시에 채워진다. 사용자가 "프로필 편집"을 누르면 편집 화면으로, "주소 관리"를 누르면 주소 목록 화면으로 빠진다.

이 흐름이 남기는 것 — 본 화면은 "내 상태의 빠른 확인 + 다음 작업의 출구" 두 가지에 집중하는 것이 가치.

삭제 예약 중

유예 기간 중인 사용자가 배너로 다시 들어옴

30일 유예 삭제를 신청해 둔 사용자가 마이페이지에 들어오면 상단에 남은 일수 배너가 보인다. "관리"를 누르면 데이터 프라이버시 화면으로 이동해 거기서 취소할 수 있다.

이 흐름이 남기는 것 — 사용자가 자기 계정의 "현재 상태"를 잊지 않게 한다. 배너 없이 두면 사용자가 30일 뒤에 갑자기 계정이 사라진 걸 알게 된다.

실수 방지

로그아웃을 잘못 눌렀다가 취소

메뉴를 훑다가 로그아웃을 잘못 눌렀다. "로그아웃하시겠습니까?" 확인 다이얼로그에서 "취소"를 누르면 화면 그대로 머문다. 로그아웃은 비가역적인 동작이라 다이얼로그를 반드시 거친다.

운영자가 알아야 할 것

마이페이지가 깨지면 사용자가 "내 상태"를 인지하지 못한다
  • 요약 카드의 값이 들어오지 않을 때 비어 있는 채로 두는 것이 정책이다. 네 개의 카드가 각자 로딩 스피너를 그리면 화면이 깜빡거리는 느낌이 강해진다. 값이 비어 있는 카드를 어떻게 운영 모니터링에서 잡을지는 별도 책임이다 — 사용자에게는 안 보인다.
  • 삭제 진행 배너는 취소 직후 즉시 사라져야 한다. "취소했는데 왜 아직 배너가 있냐"는 불신은 계정 삭제 흐름 전체에 대한 신뢰를 깬다.
  • 본 화면은 데이터를 직접 바꾸지 않는다. 하위 메뉴에서 어떤 변경이 일어났는지 본 화면이 일일이 다시 호출해 채우면 같은 화면에서 중복 호출이 누적된다. 각 도메인이 자기 책임으로 다시 채운다.
