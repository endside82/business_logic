F12-03. 카테고리별 알림 설정

카테고리(이벤트·클럽·결제·리뷰·마케팅 등) 토글로 여러 알림 type을 한꺼번에 ON/OFF. 카테고리 OFF 시 매핑된 모든 type을 일괄 PATCH.

이 기능 안의 하위 항목

동작 1

카테고리별 마스터 토글

동작 2

type별 세부 토글

동작 3

일괄 PATCH

사용자 시나리오

S1

마케팅 OFF

S2

세부 type만 OFF

S3

일부 실패 처리

서버 계약

MethodPath
GET/api/v1/notifications/settings
PATCH/api/v1/notifications/settings/{type}
