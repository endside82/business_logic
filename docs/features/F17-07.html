F17-07. 고정형 결제 (WALLET / BANK_TRANSFER)

FIXED 모임은 코스 단위로 한 번에 결제한다. WALLET은 즉시 차감 + 분개, BANK_TRANSFER는 호스트 직접입금(off-ledger)이라 분개 없고 호스트가 별도 확인해야 한다. 멤버당 동시 활성 결제 1건은 STORED 생성컬럼 + UNIQUE 인덱스로 보장한다.
