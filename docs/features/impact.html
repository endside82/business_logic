부수효과 매트릭스

한 기능을 바꿀 때 같이 흔들리는 영역을 본다. 알림·돈·캘린더·위치·신뢰 다섯 축은 도메인을 넘어 횡단으로 작동하기 때문에, 기능 PRD만 보고 있으면 같이 깨지는 부분을 놓치기 쉽다.

검토 순서

기능 변경이 있을 때 다섯 영역을 순서대로 묻는다. 한 영역이라도 “예”면 해당 영역의 정책 PRD를 함께 갱신한다.
