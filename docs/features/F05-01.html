F05-01. 키워드 검색

한 번의 검색으로 이벤트·클럽·플랜 세 도메인을 동시에 조회한다. 결과는 탭으로 분리되고 정렬은 이벤트 탭에서만 동작한다. 무한 스크롤은 이벤트만, 클럽/플랜은 첫 20건만 보여 빠른 의사 결정에 집중한다.

동시 조회
3
시나리오
8

이 기능 안의 하위 항목

동작 1

다중 도메인 검색

이벤트·클럽·플랜 3개 호출 동시 발사.

동작 2

탭 전환(추가 호출 없음)

한 번 로드된 결과를 캐시로 즉시 표시.

동작 3

정렬 + 무한 스크롤(이벤트)

추천/최신/인기/가격 순.

동작 4

idle 상태 surface

저장된 검색·최근 검색어·트렌딩 키워드.

사용자 시나리오 8개

S1

키워드 검색 → 이벤트 진입

  1. “등산” 입력 → 3 도메인 동시 호출
  2. 이벤트 탭 활성, 카운트 칩 표시
S2

탭 전환

  1. 클럽 탭 → 캐시로 즉시 표시
S3

무한 스크롤(이벤트만)

  1. 이벤트 페이지 누적
  2. 클럽/플랜은 첫 20건만
S4

정렬 변경

  1. 0페이지부터 재조회
S5

idle surface

  1. 저장된 검색·최근·트렌딩
S6

0건 결과

  1. 빈 상태 안내 + 트렌딩 키워드
S7

최근 검색어 클릭

  1. 입력란에 채워지고 즉시 재검색
S8

비로그인

  1. 저장된 검색·히스토리 숨김
  2. 검색은 가능
