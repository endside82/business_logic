반영하지 않은 것

각 기능 PRD의 본문(서버 계약·프론트 계약·Gap/Risk 등)은 이 시각화가 미러링하지 않는다. 시각화는 인벤토리와 입구 역할에 머물고, 기능 단위 source-of-truth는 02_feature_prds/의 PRD와 units/<domain>/<feature>/ 실사 자료를 직접 본다. 기능별 Trace·Risk 수치는 원문이 갱신되면 features.js를 재생성해서 동기화한다.
